# spatial-id-converter

以下の機能を有するライブラリ。

* 直方体を Cesium で読み込み可能なモデルに変換する
* 空間 ID を直方体として表現する
* 日本付近のおおよそのジオイド高を取得する


## サードパーティライセンス

以下のファイルは、サードパーティのライセンスが適用されます:

### assets/jp_gsi_gsigeo2011.tif

日本付近のジオイドモデル

https://github.com/OSGeo/PROJ-data/tree/master/jp_gsi
